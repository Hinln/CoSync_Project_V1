# 同频 (CoSync) — 界面设计方案

## 1. 设计风格

**赛博简约风 (Cyber Minimal)**，以 Indigo (#6366f1) 为主色调，搭配深色/浅色双模式。整体追求干净、现代、科技感，同时保持 iOS HIG 的原生体验。

### 色彩体系

| Token | Light | Dark | 用途 |
|-------|-------|------|------|
| primary | #6366f1 | #818cf8 | 主色调、按钮、强调 |
| background | #ffffff | #0f0f14 | 页面背景 |
| surface | #f5f5f7 | #1a1a24 | 卡片、输入框背景 |
| foreground | #11181C | #ECEDEE | 主文字 |
| muted | #687076 | #9BA1A6 | 次要文字 |
| border | #E5E7EB | #2a2a3a | 分割线、边框 |
| success | #22C55E | #4ADE80 | 成功状态 |
| warning | #F59E0B | #FBBF24 | 警告状态 |
| error | #EF4444 | #F87171 | 错误状态 |

## 3. 各屏幕内容与布局

### 3.1 登录页 (/login)

**布局（竖屏 9:16）：**
- 顶部 30%：App Logo + "同频" 标题 + 副标题 "同频共振，真实连接"
- 中部 40%：
  - 手机号输入框（带 +86 前缀，11位数字）
  - 验证码输入框 + 发送验证码按钮（60s倒计时）
  - 登录/注册按钮（Indigo 主色，圆角）
- 底部 30%：用户协议 & 隐私政策链接

### 3.2 实名认证页 (/verify)

**布局：**
- 顶部：安全盾牌 Icon（Indigo 色）+ "实名认证" 标题
- 隐私承诺卡片（带锁图标）：
  > 「隐私安全」身份信息直连公安系统核验，同频不存储您的身份证号与真实姓名。
- 手机号输入框（灰色锁定，显示注册手机号，不可编辑）
- 真实姓名输入框
- 身份证号输入框
- 认证按钮

### 3.3 动态广场 (HomeScreen)

**布局：**
- 顶部标题栏："广场"
- 瀑布流双列卡片列表：
  - 每张卡片：图片 + 文字内容 + 用户头像昵称 + 点赞数
- 右下角浮动发布按钮（FAB，Indigo 色 "+"）
- 下拉刷新 + 上拉加载更多

### 3.4 动态详情页 (/post/[id])

**布局：**
- 顶部导航栏：返回 + "动态详情"
- 用户信息行：头像 + 昵称 + 发布时间 + 性别标识
- 动态内容：图片 + 文字
- 操作栏：点赞 + 评论数
- 评论列表（支持回复）
- 底部评论输入框

### 3.5 消息列表 (MessagesScreen)

**布局：**
- 顶部标题栏："消息" + 创建群聊按钮
- 会话列表：
  - 每行：头像 + 昵称/群名 + 最后一条消息 + 时间 + 未读数角标

### 3.6 聊天页 (/chat/[id])

**布局：**
- 顶部导航栏：返回 + 对方昵称
- 消息气泡列表（左右分布）
- 底部输入栏：文字输入 + 发送按钮

### 3.7 个人中心 (ProfileScreen)

**布局：**
- 顶部用户卡片：
  - 头像（大圆形）+ 昵称 + 性别图标（锁定）+ 实名认证状态标识
- 功能列表：
  - 我的动态
  - 编辑资料
  - 实名认证（未认证时显示）
  - 设置
  - 退出登录

## 4. 关键用户流程

### 流程 A：注册登录
1. 打开 APP → 登录页
2. 输入手机号 → 点击发送验证码
3. 输入验证码 → 点击登录
4. 首次登录自动注册 → 进入广场首页

### 流程 B：实名认证
1. 点击发布动态 → 检测未实名 → 跳转实名认证页
2. 手机号自动填充（锁定）
3. 输入真实姓名 + 身份证号
4. 点击认证 → 调用阿里云三要素 API
5. 认证成功 → 根据身份证第17位设置性别 → 返回

### 流程 C：发布动态
1. 点击 FAB "+" → 检测实名状态
2. 已实名 → 进入发布页
3. 选择图片 + 输入文字 → 发布
4. 发布成功 → 返回广场

### 流程 D：私聊
1. 在动态详情页点击用户头像 → 进入聊天页
2. 输入消息 → 发送
3. 实时收到对方回复

### 流程 E：群聊
1. 消息页点击创建群聊 → 选择成员 → 创建
2. 进入群聊页 → 发送消息

## 5. 交互设计要点

- 所有按钮使用 scale: 0.97 + Haptics.impactAsync(Light) 反馈
- 列表项使用 opacity: 0.7 按压反馈
- 页面切换使用 250ms 淡入动画
- 实名认证页安全盾牌使用微动画（轻微呼吸效果）
- 输入框聚焦时边框变为 primary 色
